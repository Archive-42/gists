Website Custom Protocols

We can Register our website in the current browser as a handler for a custom protocol, like `web+cheese`.

So, any time the current browser gets a request like `web+cheese:mozzarella` then it will open the registered URL. It can pass the value after the `:` to that URL too.

A link to `web+bob:gene`

Works on Opera, Firefox, Edge, and Chrome (if using https).
