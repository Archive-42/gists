10 Great VSCode Tips

for Web Developers

  • Move a line of code up/down: ⌥ ↑|↓
  • Duplicate a line of code: ⇧ ⌥ ↑|↓
  • Add extra cursors: ⌘ ⌥ ↑|↓
  • Select next occurrence: ⌘ D
  • Select ALL occurrences: ⌘ ⇧ L
  • Code Splitting & Folding ⌥
  • Emmet built-in feature
  • “Auto Rename Tag” extension by Jun Han
  • “Live Server” extension by Ritwick Dei
  • Adjusting font sizes: ⌘ +/-
